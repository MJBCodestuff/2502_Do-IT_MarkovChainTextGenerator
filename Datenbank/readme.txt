Um die Verwendung von Datensätzen in der benötigten Größe möglich zu machen muss die my.ini von XAMP unter 
xamp/mysql/bin für eine größere erlaubte paketgröße und eine größere erlaubte Logfile konfiguriert werden.
Die mitgelieferte my.ini hat diese Einstellungen bereits, eine Sicherheitskopie vor dem ersetzen ist natürlich
trotzdem zu empfehlen.
Der XAMP server mit der mitgelieferten Datenbank muss vor Programstart aktiviert sein.
